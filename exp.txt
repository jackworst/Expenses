Expenses Tracking - Pet Project, Phase 0

Goal:
- Basic tracking and analysis tool for all personal expenses

Features:
- Offline data entry on mobile device, basic but efficient
- Synchronization with server, one-way, manually triggered
- Analysys, tabular stats for month & year
- Export to Google Docs via CSV (if supported by Google)
- Supports at least date, amount, category (fixed set), text (optional)
- Editing only via direct db access
- Server running on "mini2"

Technology:
- nodejs
- mongodb
- jquery
